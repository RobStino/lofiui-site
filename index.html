Low-fidelity Tailwind CSS components.

Headwind. You know… like a headstart on tailwind!

I'd heard a lot about utility class CSS over the years, but it wasn't until I started playing with Tailwind CSS did I fully realise how simple and scalable a methodology it was.

Headwind is my love-letter to Tailwind and other frontend devs like me. It is intentionally utilitarian and the components it includes are built to demo efficient HTML structure and a functional use of Tailwind. You may use these components as a starting point for your own components and then apply all your own amazing style and design on top.

All components adhere to 3 rules. These rules are designed to make the components as simple and accessible as possible.

  1. Just the CDN - No custom classes or stlying. Just vanilla Tailwind.
  2. NO JS - That way you can use whatever JS library or framework you prefer.
  3. Minimal Styling - Just enough styling to make sense. No more.

When you need a component to function in a particular way and be based on efficient use of HTML and Tailwind classes, I hope Headwind can help you out.

Cheers!

Rob
